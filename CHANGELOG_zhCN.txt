SoHighPlates v2.3 - 本地化增强版更新日志
=====================================================

更新日期: 2026-04-18

【新增功能】
✓ 完整的简体中文 (zhCN) 本地化支持
✓ 完整的繁体中文 (zhTW) 本地化支持
✓ 智能语言检测，自动切换界面语言
✓ 中文客户端自动使用内置中文字体（ARKai_T.ttf）
✓ 避免中文显示乱码或方块问题

【技术改进】
✓ 新增 locale.lua 本地化文件
✓ 优化字体加载逻辑，支持多语言环境
✓ 所有 UI 文本支持动态本地化
✓ 保持向后兼容性，不影响现有配置

【修改文件】
• locale.lua (新增) - 本地化字符串表
• __sNpCore.lua - 添加中文检测和字体配置
• __sNpPlate.lua - 姓名板文本使用中文字体
• __sNpOptions.lua - UI 界面本地化
• SoHighPlates.toc - 调整文件加载顺序

【兼容性】
✓ 完全兼容现有配置文件
✓ 英文客户端继续使用原有界面和字体
✓ 其他语言客户端不受影响
✓ 无需用户手动配置

【使用说明】
• 中文客户端：自动显示中文界面，姓名正常显示
• 英文客户端：保持英文界面，功能不变
• 输入 /snp 打开设置菜单查看效果

【注意事项】
⚠ 需要 WoW 2.4.3 官方客户端
⚠ 确保客户端包含内置中文字体文件
⚠ 如遇字体问题，可尝试重置插件设置
